A Window

Creative Coding Generative
Creative Coding Generative Art p5.js Interactive Web

A Window is a creative coding project that uses code as a medium for poetic expression. The work generates a dynamic, ever-changing visual scene — a window that looks out onto something different every time it is opened. Built with creative coding tools, the piece explores how algorithmic processes can simulate the organic variability of the natural world. It is both a technical exercise and a meditation on perception, presence, and the act of looking.
