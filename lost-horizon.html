Lost Horizon

Game Design Interactive Narrative
Game Design Narrative World Building Level Design

Lost Horizon is a game design project that immerses players in an atmospheric journey through landscapes caught between memory and reality. The game explores themes of displacement, longing, and the search for belonging through its environment design, sound, and player mechanics. Each level is crafted as an emotional space — visually and mechanically reflecting the inner state of its protagonist.
