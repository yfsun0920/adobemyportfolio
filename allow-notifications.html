Allow Notifications

Game Design Interaction Design
Game Design UX Critical Design Mobile

Allow Notifications is a game that uses the language and conventions of smartphone notification interfaces as its primary mechanic. By appropriating familiar UI patterns — pop-ups, permission dialogs, alerts — the game questions the relationship between users and the apps that compete for their attention. It is both a commentary on digital consent and a playful subversion of mundane interactions we accept without thought.
