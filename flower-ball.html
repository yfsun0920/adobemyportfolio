Flower Ball

Game Design Casual Game
Game Design Casual Illustration UI Design

Flower Ball is a casual game design project with a playful, nature-inspired aesthetic. The game centers around a simple but satisfying core mechanic involving blooming flowers and ball physics. The visual design draws from botanical illustration, creating a tactile, organic feel in the game's world. The project explores how a gentle, beautiful aesthetic can coexist with an engaging gameplay loop.
