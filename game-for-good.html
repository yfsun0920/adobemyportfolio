Graphic Design Game Jam Visual Identity Branding Print

Created for a Game for Good game jam, this graphic design project encompasses the visual identity and communication materials for a socially conscious game development event. The design work spans logo, branding system, promotional materials, and game jam assets. The visual language reflects the spirit of the event — collaborative, purposeful, and optimistic — communicating that games can be a force for positive change.
